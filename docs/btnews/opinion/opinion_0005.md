2023年11月13日星期一，欢迎收看第五期高见。本期，我们谈谈桦南县体育馆坍塌事故。

先说结论，体育馆屋顶坍塌是因为四道人字钢梁发生脆性断裂。断裂的原因是承载力不足，是质量问题，大学只是压垮骆驼的最后一根稻草。对于这桩事故，项目的开发商、设计、施工、监理、审核、验收单位，以及运行期间负责安全检查的部门，都负有责任。

接下来，我们顺着现场的蛛丝马迹，一步步还原事故全貌。

2023年11月6日晚7点20分，黑龙江佳木斯桦南县悦城广场南侧体育馆发生屋顶整体坍塌。事发时，南侧球场内有7人，其中3人自行脱险，4人被困。事发后，佳木斯与桦南县的消方人员赶往现场。7点50分左右，一名被困者获救。在随后的救援中，找到了其余3人，送医院抢救无效死亡。到当天半夜，现场确认无其他遇难者，救援结束。

随后，为避免次生事故发生，公安机关对事故现场进行封控。同时控制了悦城体育俱乐部负责人。视线两级纪检监察机关介入，成立调查组开展调查。事故发生后的第二天早晨，调查组进行现场实地勘察，对坍塌建筑材料、钢结构测量取样。之后将由黑龙江省寒地建筑科学研究院负责事故鉴定。根据新华社报道，坍塌体育馆是桦南县悦城广场综合体7号楼的西侧部分。

2017年7月开工，2018年8月竣工，2020年7月通过验收。建设单位是悦城房地产，施工单位是海丰伟业，监理是三江建设。悦城广场7号楼高9.05米，框架结构，分为东西两个部分。东部屋顶是钢筋混凝土现浇屋面，坍塌的西部，面积675平，屋顶是H型变截面钢梁结构。根据桦南县政府网站信息，悦城广场在2018年1月开业，事故体育馆虽然是2020年7月才通过验收，但在2019年7月就曾作为“乔氏杯”台球比赛的赛场，进一步搜索还能看到当时的赛场布置照片。当年开发、经营悦城广场的三家公司分别在2020年、2022年注销。悦城房地产法人是吴立先，事故体育馆在2019年举办台球比赛之前，吴立先曾说，建这个场馆花了1600万。台球比赛后，吴立先曾公开与参赛选手合影，一同出席比赛的县领导还有好几个。裁判文书网上可以搜到12篇与吴立先有关的文书。其中，黑0822民初1756号披露，吴立先与杨金丽是夫妻关系，黑08民中409号披露，吴立先是桦南县建设局退休干部，杨金丽是桦南县环保局干部。吴立先出生于1970年，到2018年才48岁就已经退休干部，很有意思。根据2017年报道，悦城广场项目由北京得益音通投资3.8亿建设。项目用地来自奥林新村37平湖改造，在县政府网站中也可以查实这个地块是2015年拆迁计划的一部分。事故发生后，南方都市报联系了投资方北京得益音通，发现总经理翟士才也是桦南当地人。到这里，项目招商隐私的过程基本可以还原。棚改政策出台后，桦南县在2015年拆迁了奥林新村，随后建设局干部，当年45岁的吴立先办了退休，找到熟人翟士才要了一大笔项目投资，摇身一变成为开发商。这是个很有90年代风格的故事。项目背景方面，已经有不少媒体挖掘过了，本期不多说，我们继续看事故细节。

事故体育馆叫悦城广场全民健身体育馆，部分媒体把这座体育馆与桦南县全民健身中心搞混了。桦南县全民健身中心的屋顶是横架结构，与事故体育馆的H型变截面钢梁结构不一样。根据事故体育馆过去的视频可以看到，体育馆主体结构是混凝土柱加人字钢梁。与常见门式钢结构相似，体育馆屋顶在人字梁之间安装了檩条与拉条，盖上屋面板，钢梁架在混凝土柱顶端，山墙处没有安装钢梁，靠山墙支撑。屋脊设有一道檩条，屋脊两侧各有八道檩条。根据这些信息，我们可以在航拍图像基础上，粗略画出原有的屋顶结构。檩条间距一般在1.5米以内。新华社说这座场馆面积是675平，大致可以推断平面尺寸是32米乘22.5米。结合另一个角度的俯拍照片，可以大致推断建筑较长一侧的柱间距，也就是人字梁之间的间距。在这个基础上结合航拍照片就能发现很多事故细节。首先，四道人字钢梁都是从中间发生断裂。右侧断裂的两道人字梁基本保持了原有朝向，而左侧两道人字梁在断裂后朝向有比较夸张的偏离。此外，现场右侧的檩条有比较平缓的弯曲，符合压弯的特征，而左侧有一个折成直角的人字梁，应该是坍塌瞬间造成的剧烈形变。左侧山墙挂着几个屋面盖板，露出了盖板侧面构造。左侧地面上还有一个砸弯的盖板，只剩下钢筋网，可以看出的典型免拆模版混凝土楼盖。这种盖板，下部是保温板，中间是钢筋网片，上面浇筑混凝土。混凝土盖板很重，所以在现场救援时才会出现几个人都很难搬动一个盖板的情况。为了方便大家理解，我简单建了个模型。

事故发生后，新京报采访的当事人，披露了大量细节。根据新京报采访，当日下午，当地居民听到了雷声，这是雷达雪现象，是暴雪的征兆。当晚7点多，在体育馆内打球的小明发现，房子顶层上面爆皮，一直往下掉墙皮。这应该是檩条被压弯后，屋面盖板发生变形，盖板底部保温板涂层开裂脱落。7点20分，现场响了蹦蹦蹦三声，应该是人字梁发生的脆性断裂。小明在往外跑时出现了第一次坍塌，之后他缩到一个柱子旁边，发生了第二次坍塌。第二次坍塌后小明跑出体育馆，之后又与其他三人返回现场。在7点50分救出一名被困人员。四人准备再次返回现场救人时，发生了第三次坍塌。之后救援人员赶到现场开展救援作业。要还原事故过程，有两点需要搞清楚。一是小明是从哪个门逃出去的。二是屋顶三次坍塌的顺序。从航拍图来看，体育馆西侧没有疏散口，东侧有两小一大三个门，南侧有个开口，看上去像个门。但另一张室外角度的照片证明，这堵墙没有门。开口位置从室内视角看是放篮球架的位置。至于体育馆北侧，从其他视角也可以看到没有门。也就是说，整座体育馆只有东侧有三个门。根据过去体育馆内的视频可以看到，东侧这三个门，较大的门是出入口，通往7号楼东侧区域，而另两个较小的门，里面还挂着衣服，应该是更衣室和卫生间。南侧篮球架的位置，之所以会出个缺口，是因为这座体育馆原本预留了大量窗洞，后来用保温板石膏板封堵了，所以才会在屋顶坍塌的过程中冲破几个缺口。换句话说，整座体育馆有且仅有一个出入口，还不是直通室外的，完全不符合消方疏散规定。就算没有这次坍塌事故，一旦发生火灾，出入口上方的防火卷帘一落下来，体育馆里的人就被彻底困住了。正因为整座体育馆高度封闭，所以坍塌时的巨大吸压会顺着体育馆唯一的开口，将东侧建筑的窗户都给冲破了，这就是为什么周围住户听着像发生了爆炸。明确了出入口之后，我们进一步还原坍塌过程。为了方便理解，我把每个人字梁都编上了编号。小明跑到逃生出口时发生了第一次坍塌，他将同伴推出去之后，被建筑材料挡住了去路，随后发生了第二次坍塌。小明说，他是根据地理课学到的知识，找个三角形的地方。现场最符合这个特征的，就是出口防火卷帘挡板下方，靠近柱子的区域。根据事故后照片来看，出口左侧角落相对安全，应该是小明躲避的位置。小明将同伴推出去之后，三号钢梁与盖板砸下来，小明躲在角落里，没有被钢梁盖板砸到，但被砸掉的防火卷帘挡板擦伤了，紧接着就发生了第二次坍塌。第二次坍塌后，小明看没有继续掉东西，就跨过建筑材料逃了出去。在门式钢结构中，居中的人字梁中心受到的弯矩最大，这次坍塌的体育馆屋顶，在左右两侧是靠山墙支撑，所以12号钢梁与78号钢梁的受力情况，要好于中间的34号钢梁与56号钢梁。结合出口位置可以确定，第一次坍塌就是中间的34号钢梁发生了断裂掉落。结合事故后钢梁的位置，56号钢梁都明显朝左偏，应该是在34号钢梁断裂掉落之后，56号钢梁才发生断裂。紧接着在与78号钢梁之间的檩条拉扯下，断口像钟摆一样甩到了左边，靠外的12号钢梁，距离右侧山墙有6.6米，而78号钢梁距离左侧山墙距离只有5.4米，所以右侧的12号钢梁受力情况比左侧的78号钢梁要差。因此，12号钢梁的断裂坍塌应该比左侧的78号钢梁要早。从事故后的掉落位置来看，应该是第二次坍塌的部分。第二次坍塌后，小明回头救了一个人，之后发生了第三次坍塌，完全没办法继续救人了。根据救援现场的视频来看，救援区域集中在左上角。这再次说明最后坍塌的应该是左侧钢梁。由于逃生口偏右，左侧区域最不容易逃生，事故现场只有左侧山墙挂着7块盖板，其余盖板都掉落在地上，这说明最左侧屋面有个缓慢的形变过程，等到最终坍塌时，下落的动能已经不足以将盖板从山墙上拽脱。结合事故救援现场与事故后的航拍可以看到，78号钢梁的7号部分发生了夸张的扭曲，钢梁发生了90度扭转。这也证明了左侧区域有过去一次缓慢的形变过程，将78号钢梁像拧麻花一样拧了90度，5号与7号钢梁之间还挂着一块盖板，也证明这个形变过程很缓慢，屋面防水层的破快细节也能证明这一点。体育馆东侧也就是画面下方区域的屋顶安装了招牌字。这些字有一部分支撑点就在屋面防水层上，靠左侧的几个字都被向里拽到，而右侧的字没有被拽到，说明右侧区域的坍塌都是瞬间坍塌，防水层被瞬间垂直撕断，而左侧防水层受到了水平方向的缓慢拉拽，整个防水层都被抽走了。到这里，我们基本上已经可以完整还原坍塌过程。为好晚上7点20分，12号、34号、56号钢梁连续发生脆性断裂，发出蹦蹦蹦的三声，紧接着34号钢梁连同上面的盖板一起坍塌。几分钟后，12号钢梁也连同上面的盖板一起坍塌。56号钢梁断裂后，与78号钢梁在檩条拉扯下藕断丝连，悬在半空中，与滑落的盖板将左侧区域封闭。半个小时后，78号钢梁不堪重负连同上方盖板一起坍塌。救援人员发现最后3名被困学生的具体位置目前还不清楚，如果是在7号钢梁下方，这意味着被困学生很可能一开只是被封闭在左侧空箱里。结果这个体育馆只有一个出入口还是在右侧。被困学生完全找不到出路，只能看着上方残留的屋顶在长达半个小时的时间里缓慢变形，不断挤压着剩余空间。最后在8点左右，残留的屋顶整体坍塌下来。

在具体分析事故原因之前，首先可以确定，积雪不是核心原因，只是压垮屋面的最后一根稻草。

桦南县6号降雪是特大暴雪，24小时降雪量44毫米，积雪深度32厘米，刷新了当地11月的历史记录，但没有打破最大降雪记录。对比2号早晨和7号早晨的画面，可以看到东侧建筑屋顶是个简修屋顶，属于不商人物面。女儿墙高度600毫米左右，扣除屋面保温层、找平层、防水层这些构造，女儿墙高出屋面表面大概30-40厘米。到7号早晨，只有部分角度区域的积雪，勉强与女儿墙平齐，也就是说6号晚上体育馆屋顶坍塌前，屋面积雪厚度最多也就是30多厘米。6号降雪是雷达雪，意味着高空有较强的暖湿空气，所以雪中还有冰，比一般的降雪密度要大一些。往高了给他算，就当是一平米40公斤。建筑结构荷载规范第7.2.2条强制规定了雪压设计指标。根据数据表，佳木斯地区50年一遇的雪压设计荷载是每平米0.85千牛，相当于每平米86.7公斤积雪。即便是按照临时建筑算，选用10年一遇的雪压荷载设计值，也是每平米0.6千牛，也就是说佳木斯地区的临时建筑屋顶也要能承住每平米61.2公斤的积雪。此外，建筑结构在实际设计时还会留有余量，确保在遇到地震、偶然荷载的情况下也安全。无论如何，这些积雪都不能作为事故的理由。积雪还远没有达到设计指标，屋顶就坍塌了。导致这一结果的原因，可能是设计不足，施工偷工减料或不规范，也可能是发生了严重锈蚀，没有及时维护。这座体育馆竣工到现在才五年，从体育馆几周前的视频来看，人字梁与檩条的漆面也正常，没有出现严重锈蚀。

根据当事人的说法，坍塌之前发生了三声巨响，符合钢结构脆性断裂的特征。高寒地区焊接时，因为环境温度低，焊接后的钢材冷却速度比其他地区更快，焊件变形和裂纹的问题更严重，更容易出现脆性破坏事故。魔鬼隐藏在细节里，我们重新再看一遍事故前的体育馆内部视频。放大后可以看到，人字梁出现了明显的下弯，尤其是2号与4号钢梁区域，下弯最严重。刚好对应了最先发生坍塌的右侧。一开始我还以为是拍摄的问题，但随后我又找了另一个人拍摄的视频，发现钢梁确实下弯了。这两个视频的拍摄时间，一个是几周前，另一个是在去年2月份，都能看出来钢梁明显下弯。说明结构的变形损坏早就发生了。到今年秋天的时候，这座体育馆的屋顶结构已经在崩溃边缘，所以遇到大雪就立刻发生了坍塌事故。20多米跨度的钢梁不可能是从厂家一道运到现场的，通常都是在钢材厂做好分段，送到现场焊接。一旦焊接有缺陷，就会大幅削弱结构的可靠性。从事故现场来看，四组人字钢梁都是居中断裂，很可能中间部位就是焊接的。体育馆在2018年8月竣工，我在网上找到了2017年9月的现场照片，焊接作业的具体时间很可能是2017年深秋或2018年开春。环境温度低，所以钢梁施工出现了隐患。分析到这一步，似乎事故原因只是施工焊接问题。先别急着下结论，我们继续看细节。

网络上很容易找到屋顶结构相似的篮球馆，但这些篮球馆往往都是选用完整的门式钢框架结构，柱子也是钢柱，与钢梁钢性铆接在一起。这样一来，在屋顶荷载作用下，钢柱也能承担一部分弯矩，即便钢梁出现断裂倒塌，钢柱也能拉住钢梁一侧，构造一个三角区，不会让整个屋面垂直坍塌下去。而事故体育馆的结构是混泥土柱加钢梁，这种结构形式很难将钢梁与混泥土结构进行钢性连接。通常只能用螺栓角接，起到一点固定效果。体育馆南北两侧没有钢梁，是靠山墙支撑屋面。根据建筑抗震设计规范第9.1.7条，厂房同一结构单元内，不应采用不同的结构形式。厂房端部应受物价，不应采用山墙承重。也就是说，这座体育馆在设计上违反了抗震规范。由此可见，这个项目在节能用钢方面花了很多心思。国内很多建筑设计院不具备钢结构细化设计能力，往往只能提供基本方案，要靠专业的钢结构设计单位或厂家来进行设计深化。要是遇上野蛮施工，完全可能出现现场凭经验结省用钢的情况。进一步学减结构的可靠性，导致设计上先天不足。熟悉钢结构的观众，可以尝试建模计算一下，看看钢梁与檩条是不是还有其他问题。

总结一下，体育馆在设计方面疏散不符合规范，抗震不符合规范，钢结构设计可能先天不足。施工方面，焊接作业可能存在重大隐患。在项目存在明显违规的情况下，图纸审查、现场验收却都能通过。在事后运行的几年期间，没有进行任何整改。没有认何人注意到钢梁出现明显下弯。出现事故，不是一个环节出了问题，是每个环节都有问题。此外，从2017年9月的施工现场照片还可以看到，当时体育馆结构框架已经建好了，但少了一段柱子，说明这个柱子是后接上去的，不知道是什么原因。好，本期节目到此结束，感谢大家收看，我们下期再见。
